## Contenido del repositorio

Este repositorio recoge los materiales utilizados para el análisis de lo ominoso en un corpus de cuentos de la Edad de Plata.

### Carpetas y archivos

- `corpus/`  
  Corpus por autores, obtenido del repositorio https://github.com/cligs/textbox/tree/master/spanish/cuentos-espanoles. Contiene los textos completos de los cuentos.

- `scripts/`  
  Scripts de Python empleados para el análisis. Se describen de forma breve en el nombre del archivo.

- `frecuencias/`  
  - `frecuencia_palabras.csv`: tabla de frecuencia de todas las palabras del corpus.  
  - `frecuencia_palabras_ominosas.csv`: tabla de frecuencia de palabras ominosas.

- `stopwords/`  
  Lista de palabras vacías (`stopwords`). Fuente: https://github.com/Alir3z4/stop-words/blob/master/spanish.txt.

- `heurist/`  
  CSVs generados desde Heurist con información de autores y cuentos.

- `diccionario_ominoso/`  
  Diccionario ominoso limpio, preparado para el análisis.

---

## Uso

Los materiales se incluyen con fines académicos y de reproducibilidad del trabajo. Cada script de Python incluye comentarios breves sobre su función.
